## MNISTの学習ジョブを投入する

まずはマニュフェスト(yamlファイル)を作成します。
```
$ vim test.yaml
```
test.yaml
```
apiVersion: v1
kind: Pod
metadata:
 name: pytorch-job
spec:
 containers:
  - name: pytorch-container
    image: 192.168.100.xxx:xxxx/library/pytorch:20.09-py3
    command: ["/bin/sh"]
    args: ["-c", "python /workspace/examples/upstream/mnist/main.py"]
    resources:
      limits:
        nvidia.com/gpu: 1
 imagePullSecrets:
    - name : nvidia
 restartPolicy: Never
```

マニュフェストからジョブを投入する
```
kubectl create -f test.yaml
```

ジョブの確認
```
$ kubectl get pod
```
